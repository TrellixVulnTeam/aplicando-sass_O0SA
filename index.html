Bienvenidos a SASSS

Texto instructivo para Instalar SASS y crear Variables

Recordar que es necesario instalar Node JS. Una vez hecho eso continuamos..

Es necesario presionar la tecla "Enter" luego de realizar la acción de cada item.

  1. Abrir VSC y seguidamente la Consola Terminal (powershell).
  2. En la terminal escribír: node -v (Comprobamos version de Node).
  3. Escribir: npm init
  4. Presionar enter en forma "metralleta" (8veces).
  5. Escribir: Yes
  6. Escribir: npm install node-sass nodemon (Se nos creará el package.json).
  7. Ingresar en "package.json".
  8. Copiar las siguientes dos lineas de Código, dentro de "scripts": { (Presionar Enter aquí).
  9. "build-css": "node-sass --include-path scss scss/prueba.scss css/prueba.css", "watch-css": "nodemon -e scss -x \"npm run build-css\"",
  10. En VSC crear carpeta con nombre: "SCCS"
  11. Crear hoja con nombre "prueba.scss" (Dentro de carpeta "SCSS").
  12. Crear carpeta "CSS"
  13. Dentro de esta, crear hoja de estilo con nombre "prueba.css"
  14. Pararse nuevamente en la terminal y escribir: npm run watch-css
  15. Dentro de carpeta "SCSS" crear carpeta: "partials"
  16. Crear hoja "_header.scss" (Dentro de carpeta "partials").
  17. Agregar: @import "partials/header";
  18. Usa tu imaginacion para crear variables con el simbolo "$" y haz que tu código sea reautilizable.
  19. Para realizarse el preprocesado de SASS, la terminal siempre debe estar abierta.
  20. NOTA: Si cerraste la terminal, volvé a ejecutár el comando: npm run watch-css (Item 14).
